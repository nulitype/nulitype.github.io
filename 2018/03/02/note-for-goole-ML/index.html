机器学习系统

通过学习如何组合输入信息，对数据作出预测。

标签和特征

标签是指要预测的真实事物：y

  • 基本线性回归中的 y 变量
  • 例如未来的房价，图片显示的物种

特征是指用于描述数据的输入变量：x

  • 基本线性回归中的 {x1, x2, …, xn} 变量
  • 简单的机器学习项目可能只会用到单个特征，但是复杂的项目可能会用到数百万个特征
  • 比如在垃圾邮件检测器中，特征可以包括：特定短语，发件人地址，发送时间等信息。

样本和模型

样本是指数据的特定实例：x
有标签样本具有 {特征， 标签} : (x, y)

  • 用于训练模型
    无标签样本具有 {特征, ?} : (x, ?)
  • 用于对新数据作出预测
    模型可将样本映射到预测标签：y’
  • 定义了特征与标签之间的关系。（垃圾邮件检测模型会把某些特征与“垃圾邮件”这个标签联系起来）
  • 训练表示创建或学习模型。向模型展示有标签的样本，让模型逐渐学习特征与标签之间的关系
  • 推断表示将训练后的模型应用于无标签样本。也就是说使用训练后的模型做出有用的预测

回归与分类

回归模型可预测连续值。例如房价，广告点击率
分类模型可预测离散值。比如说照片显示的动物

线性回归

对于一条直线可以简单地写出一元函数 y = kx + b ，y 是要预测的值，k 是斜率, x 是输入特征的值， b 是 y 轴截距。
按照机械学习的惯例，会把方程式写为：
y' = b + w1x1

其中 y' 是指预测标签（理想输出值），b 指的是偏差（在一些机械学系的文档中也被称为w0），w1是指特征1的权重，x1是指特征。

具有多个特征的模型可以采用：
y' = b + w1x1 + w2x2 + ... + wnxn

训练与损失

训练模型表示通过有标签样本学习所有权重和偏差的理想值。
在监督式学习中，机器学习算法通过：检查多个样本并尝试找出可最大限度减少损失的模型。此过程称为经验风险最小化。

损失是一个数值，表示对于单个样本而言，模型预测的准确度。训练模型的目标是从所有样本中找到一组平均损失较小的权重和偏差。
